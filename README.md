## 项目说明

#### 当前版本v1.5.0
增加登陆/注册页面 对接MongoDb数据库

本项目是在个人学习Nodejs开发的聊天室项目，技术使用njs模版和socket.io两个第三方库以及MongoDb数据库

### 项目使用

```js
npm install socket.io
npm install njs
```

或者

```js
npm i
```
