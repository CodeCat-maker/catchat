## 项目说明

#### 当前版本v1.1.1
修复消息框和用户框溢出的问题

本项目是在个人学习Nodejs开发的聊天室项目，技术使用njs模版和socket.io两个第三方库

### 项目使用

```js
npm install socket.io
npm install njs
```

或者

```js
npm i
```
